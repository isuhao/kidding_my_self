The manual

These are the pages by suggested reading order:

  • Basic programming in QVision:
    • Hello world. This section details how to create a simple QVision application from zero.
    • The QVImage class section explains the basics of the QVision class for image objects.
    • The Image and video input/output section reviews the image and video input/output functionality.
    • The Image processing functionality describes the main image processing functionality included with the QVision.
    • Section Matrices and vectors reviews the most common operations available for matrices and vectors in the QVision.
  • Interoperability: using the QVision with other libraries.
    • The Interoperability with the GSL library section details how to interoperate with functions of the GSL, with the QVision data types.
    • The Interoperability with the IPP library section details the image processing functionality based on the IPP library.
    • The Interoperability with OpenCV section describes how to use functionality from the OpenCV library in a QVision application.
    • The Interoperability with Octave section indicates how to use functions from the Octave library with the QVision matrices and vectors.
  • Block oriented application development:
    • The Creating the first block-oriented application explains how to create the first simple block oriented application.
    • An advanced block-oriented example. This section offers an advanced block programming example with multiple processing blocks.
    • Creating custom processing blocks section describes how to create custom processing blocks, and add them to the block-oriented application.
    • Section Block synchronization reviews the block synchronization techniques in detail.
    • Graphical user interface. This section lists QVision input/output graphical blocks.
    • Command line parameters. This section details command line parameters of block-oriented applications.
    • The Designer GUI section explains the usage of the designer GUI. A visual block-oriented application builder.
